# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Optimus, please report it privately — do not open a public GitHub issue.

- **Preferred:** [GitHub Security Advisories](https://github.com/staqsIO/optimus/security/advisories/new) on `staqsIO/optimus`.
- **Alternative:** email [eric@staqs.io](mailto:eric@staqs.io) with details.

Please include as much of the following as you can:

- A description of the vulnerability and its potential impact
- Steps to reproduce (proof of concept, if available)
- Affected version(s) or commit(s)

## Response

This project is maintained by a small team. We make a best-effort attempt to respond within **7 days** of a report. There is currently no bug bounty program.

## Supported Versions

Only the `main` branch is supported. There are no maintained release branches — please report vulnerabilities against the latest `main`.
